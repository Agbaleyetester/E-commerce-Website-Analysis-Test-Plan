# Initial Analysis & User Workflows  

**Task:** Familiarize yourself with the Sauce Demo site and document its key user workflows.   
**Example Workflow:**  

-   **Workflow:** Successful Purchase
-   **Steps:**
    1.  User navigates to the login page.
    2.  User enters valid credentials ("standard_user", "secret_sauce") and clicks "Login".
    3.  User adds the "Sauce Labs Backpack" to the cart.
    4.  User clicks the shopping cart icon.
    5.  User clicks "Checkout".
    6.  User fills in their information (First Name, Last Name, Zip Code).
    7.  User clicks "Continue".
    8.  User clicks "Finish".
    9.  User sees the "Thank you for your order!" confirmation message.
 
-   **Workflow:** Sorting of Products
-   **Steps:**
    1.  User navigates to the login page.
    2.  User enters valid credentials ("standard_user", "secret_sauce") and clicks "Login".
    3.  User clicks the Sort icon.
    4.  User click on Name(A - Z) or Name(A -Z), it sorts the Products based on Name Alphabetically.
    5.  User click on Price(Low - High) or Price(High - Low), it sorts Prodcts based on Price from Lowest to highest and Highest to lowest respectively.

 -   **Workflow:** Successful LogOut
 -   **Steps:**
    1.  User navigates to the login page.
    2.  User enters valid credentials ("standard_user", "secret_sauce") and clicks "Login".
    3.  Navigate to Kebab icon.
    4. User click on the "LogOut".
    5. User is successfully LogOut.
    
-   **Workflow:** Successful Remove Products from Cart
-   **Steps:**
    1.  User navigates to the login page.
    2.  User enters valid credentials ("standard_user", "secret_sauce") and clicks "Login".
    3.  User adds the "Sauce Labs Backpack" to the cart.
    4.  User clicks the shopping cart icon.
    5.  User click on "Remove".
    6.  Product(s) is successfully remove from Cart.
